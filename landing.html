Explanation of the project

The purpose of this project is to analyze weather patterns in locations at varying distances from the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, I used Matplotlib to plot weather, latitude, and other variables, including: temperature, cloudiness, wind speed, and humidity. The site provides the source data and visualiztions created as part of the analysis, as well as explanations and descriptions of any trends and correlations discovered.
